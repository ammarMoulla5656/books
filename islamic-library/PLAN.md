# خطة تصميم المكتبة الإسلامية الذكية

## نظرة عامة على المشروع
مكتبة إسلامية شيعية مدعومة بالذكاء الاصطناعي، تحتوي على كتب مقسمة حسب التصنيفات مع إمكانيات بحث متقدمة وميزات ذكية.

## المرحلة الثانية: تصميم الواجهة

### 1. الصفحة الرئيسية (index.html)
- **الهيدر (Header)**
  - شعار المكتبة
  - شريط البحث السريع
  - زر تبديل الثيم (داكن/فاتح)
  - قائمة التنقل

- **قسم البطل (Hero Section)**
  - عنوان ترحيبي
  - وصف مختصر
  - زر "ابدأ القراءة"

- **التصنيفات الرئيسية**
  - فقه
  - عقائد
  - أصول الفقه
  - تفسير
  - حديث
  - تاريخ
  - أخلاق
  - أدعية

- **الكتب المميزة**
  - عرض شبكي للكتب
  - صورة الغلاف
  - اسم الكتاب
  - اسم المؤلف
  - عدد الفصول

- **الفوتر (Footer)**
  - روابط سريعة
  - حقوق النشر

### 2. صفحة التصنيف (category.html)
- عنوان التصنيف
- عدد الكتب
- فلترة وترتيب
- شبكة الكتب
- تصفح الصفحات

### 3. صفحة الكتاب (book.html)
- **الشريط الجانبي**
  - فهرس الكتاب
  - الفصول والأقسام
  - التنقل السريع

- **منطقة القراءة**
  - عنوان الفصل/القسم
  - المحتوى النصي
  - أزرار التنقل (السابق/التالي)

- **شريط الأدوات**
  - تكبير/تصغير الخط
  - تلخيص الصفحة (AI)
  - شرح النص المحدد (AI)
  - بحث داخل الكتاب
  - إضافة علامة

- **نافذة AI المنبثقة**
  - عرض التلخيص
  - عرض الشرح
  - إغلاق

### 4. صفحة البحث (search.html)
- حقل البحث الرئيسي
- فلاتر البحث
  - حسب التصنيف
  - حسب المؤلف
  - حسب الكتاب
- نتائج البحث
- البحث الذكي (تذكر جزء من النص)

## المرحلة الخامسة: ميزات الذكاء الاصطناعي

### 1. تلخيص الصفحة
- زر "تلخيص" في شريط الأدوات
- يأخذ محتوى الصفحة الحالية
- يعرض ملخصاً في نافذة منبثقة
- (واجهة جاهزة للربط مع API)

### 2. شرح النص المحدد
- تحديد نص معين
- ظهور زر "اشرح"
- عرض الشرح في نافذة
- (واجهة جاهزة للربط مع API)

### 3. البحث الذكي
- كتابة ما تتذكره من النص
- البحث في جميع الكتب
- عرض النتائج المتطابقة
- (واجهة جاهزة للربط مع API)
